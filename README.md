# 📊 Retail Consumer Behavior Analysis

## 🔍 Project Overview

This project analyzes retail transaction data to understand customer purchasing behavior, identify key sales drivers, and generate actionable business insights using Python, SQL, and Power BI.

---

## 🎯 Business Objective

* Understand customer segments and buying patterns
* Identify factors influencing purchase decisions
* Support data-driven decision making for sales and marketing strategies

---

## 🛠️ Tools & Technologies

* **Python (Pandas, NumPy)** – Data cleaning and transformation (ETL)
* **SQL** – Data analysis using JOIN, GROUP BY, and aggregation functions
* **Power BI** – Data visualization and interactive dashboard

---

## ⚙️ Project Workflow

1. **Data Collection**

   * Retail transaction dataset

2. **Data Cleaning & Transformation (ETL)**

   * Handled missing values
   * Removed duplicates
   * Standardized data formats

3. **Data Analysis (SQL)**

   * Customer segmentation
   * Revenue analysis
   * Purchase behavior analysis

4. **Data Visualization (Power BI)**

   * Sales trends over time
   * Customer demographics
   * Revenue and KPI tracking

---

## 📈 Key Insights

* Discounts significantly influenced customer purchase behavior
* Certain product categories contributed the highest revenue
* Seasonal trends impacted overall sales performance
* Repeat customers generated a major portion of total revenue
