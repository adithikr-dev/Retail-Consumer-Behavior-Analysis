# 📊 Retail Consumer Behavior Analysis

---

# 🔍 Project Overview

This project focuses on analyzing retail transaction data to understand customer purchasing behavior, identify key sales drivers, and generate actionable business insights using Python, SQL, and Power BI.

---

# 🎯 Business Objective
- Understand customer segments and buying patterns
- Identify factors influencing purchase decisions
- Support data-driven marketing and sales strategies

---

# 🛠️ Tools & Technologies
- Python (Pandas) – Data cleaning & transformation
- SQL – Data analysis using JOIN, GROUP BY, aggregations
- Power BI – Interactive dashboard & visualization

---

# ⚙️ Project Workflow
1. Data Collection – Retail transaction dataset
2. Data Cleaning (ETL)
  - Handled missing values
  - Removed duplicates
  - Standardized data formats
3. Data Analysis (SQL)
  - Customer segmentation
  - Revenue analysis
  - Purchase trends
4. Visualization (Power BI)
  - Sales trends
  - Customer demographics
  - KPI tracking

---

# 📈 Key Insights
- Discounts significantly increased purchase frequency
- Certain product categories contributed highest revenue
- Seasonal trends impacted customer buying behavior
- Repeat customers generated a major portion of revenue
