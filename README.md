# Retail-Consumer-Behavior-Analysis

---

# 🔍 Project Overview
This project analyzes retail transaction data to understand customer behavior, purchasing trends, and key business drivers using Python, SQL, and Power BI.

---

# 🛠️ Tools & Technologies
Python (Pandas)
SQL
Power BI

---

# ⚙️ Key Steps
- Performed ETL operations to clean and transform raw retail data
- Conducted SQL analysis using JOIN, GROUP BY, and aggregations
- Identified KPIs like revenue trends, customer segments, and seasonal patterns
- Built an interactive Power BI dashboard for visualization
  
---

  # 📈 Key Insights
- Discounts significantly influenced purchase frequency
- Certain product categories generated higher revenue
- Seasonal trends impacted sales performance
